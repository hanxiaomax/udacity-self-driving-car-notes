# 2.2 变量命名

对于程序员来讲，写代码最大的挑战之一，就是给变量命名，一个好的变量名，胜过千言万语，而要想起一个好名字，还真没你们想象的那么简单

基本要求：

  1. 只能在变量名称中使用常规字母、数字和下划线。不能包含空格，并且需要以字母或下划线开头。

  2. 不能使用保留字或内置标识符，它们在 Python 中具有重要含义，你将在整个这门课程中学到这些知识。python 保留字列表请参阅此处。创建对值清晰描述的名称可以帮助你避免使用这些保留字。下面是这些保留字的简要表格。

风格要求：

  1. 使用小写字母和下划线来组成变量名
  2. 尽管下划线可以作为开头，但是下划线是开头是有特殊含义的，不建议在常规情况下使用，详见：
  3. 尽管大写字母的变量名也是合法的变量名，但是一般只在定义“枚举”的时候才用
  4. 不要使用拼音
  5. 如果使用缩写，要是业内熟知的缩写，不要自创

# 3. 基本数据结构

在我们深入了解python的基础数据结构之前，首先给大家隆重的介绍两个基本函数，可以帮我们更好的学习下面的内容。

  1. print()函数。俗话说，没有什么bug是一条print解决不了的，如果有，就多加几条print。当大家在学习python的过程中有任何疑问的时候，一定要自己先尝试去定位问题，最基本的方法就是用print()函数把你感到有疑问的变量打印出来

  2. type()函数，这个函数的作用是告诉我们某个变量的类型，比如：

    s = "hello world"
print(type(s))
>>>str #知道s是string类型了

# 3.1 整数和浮点数 

  • 整型（int）是所有编程语言里面做常见到的数据类型，1，200，1024都是整形，在Python里面我们可以通过下面的方式来创建整型

    age = 10  
>>age is 10 , type is <type 'int'>
age = int(10)
>>value is 10 , type is <type 'int'>
count=int()
>>value is 0 , type is <type 'int'>

    money = 5.1 
>>age is 5.1 , type is <type 'float'>
money = float(2.2)
>>value is 2.2 , type is <type 'float'>
count=float()
>>value is 0 , type is <type 'float'>

# 3.2 布尔类型

1.布尔类型其实就是True或者False，比如我们可以给`一个变量赋值

resullt = True
print(type(reslut))
>>>bool #result的类型是bool类型

这里千万不要把True或者False当作是一个单词，然后用引号扩起来，在python中True或者False就是一个bool类型的变量

# 3.3 比较运算符和逻辑运算符

  • python中可以使用>,<,>=,<=,==,!=作为比较运算符，值得注意的是==是判断是否相等，而不是用等于号=，这点新手容易犯错误。
  • 比较运算符返回的结果是布尔类型的，也就是只有True或者False，即等式或不等式是否成立，成立为True，不成立为False
  • Python中还有三个逻辑运算符，and,or,not，分别表示与或非。
