# 迁移学习

The Four Main Cases When Using Transfer Learning Transfer learning involves taking a pre-trained neural network and adapting the neural network to a new, different data set.

Depending on both:

the size of the new data set, and the similarity of the new data set to the original data set the approach for using transfer learning will be different. There are four main cases:

new data set is small, new data is similar to original training data new data set is small, new data is different from original training data new data set is large, new data is similar to original training data new data set is large, new data is different from original training data
